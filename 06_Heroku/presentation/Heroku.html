What is Heroku?

  • a company offering cloud computing services
  • specifically Platform as a Service (Paas)
  • allows customers to develop, run and manage Web applications without the complexity of building and maintaining the infrastructure

Heroku website

PaaS

  • original intent of was to simplify the code-writing process for developers
  • a public cloud service from a provider
  • consumer controls software deployment and configuration settings
  • provider provides the networks, servers, OS, storage etc. to host the consumer's application

Heroku vs. Amazon Web Services (AWS)

  • Heroku offers Platform as a Service
  • AWS offers Infrastructure as a Service (IaaS)
  • AWS started offering PaaS via it's Elastic Beanstalk after 2011

SaaS vs PaaS vs IaaS

history of Heroku

  • one of the first cloud platforms, in development since June 2007
  • initially supported only the Ruby programming language
  • today supports several programming languages (Java, Node.js, Scala, Clojure, Python, PHP, Perl)
  • was acquired by Salesforce.com in 2010
  • base operating system is GNU/Linux (Debian or Ubuntu)

etymology

  • hero + haiku
  • a tribute to Heroku's Ruby chief architect Yukihiro Matsumoto
